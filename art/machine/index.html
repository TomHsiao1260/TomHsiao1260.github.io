Drawing machine

I built a drawing machine which successfully painted my first self-portrait (test experiment)

我嘗試獨立搭建一個繪圖機器的原型，並完成第一幅以機器作為媒材的「自畫」像。作畫過程中，只要輸入一張圖片搭配一些預處理，機器就能將畫筆定位在指定的位置並繪製想要的圖案。技術上，用的是CNC三軸定位座標(2個步進馬達＋1個伺服馬達)，並搭配Gcode作為軟體控制。此外，在這些初步嘗試後，希望能讓機器對圖像有更深層的認知 (AI)，探索人類與機器共同協作的更多可能性。
